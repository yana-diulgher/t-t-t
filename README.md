GitHub Pages - https://yana-diulgher.github.io/t-t-t/
GitHub Pages — это бесплатный сервис хостинга для статических сайтов, позволяющий публиковать веб-сайты, созданные из файлов HTML, CSS и JavaScript, прямо из вашего репозитория на GitHub. 
 GitHub Pages предоставляется бесплатно каждому пользователю GitHub . Он доступен в публичных репозиториях GitHub при наличии учётной записи GitHub Free, а также в публичных и приватных репозиториях GitHub при наличии учётной записи GitHub Pro, GitHub Team или GitHub Enterprise.


 
Netlify - soft-treacle-c48164.netlify.app
Netlify не является полностью платным сервисом — он предлагает щедрый бесплатный тариф (Starter) с достаточными лимитами для развертывания статических сайтов и небольших проектов, включая 100 ГБ трафика и 300 минут сборки в месяц. Вы можете бесплатно развернуть неограниченное количество статических сайтов без ввода данных кредитной карты. 


Vercel - https://t-t-t-one.vercel.app/
Как долго я смогу пользоваться Vercel бесплатно?
Поскольку тарифный план Hobby относится к категории бесплатного, он не предусматривает циклов оплаты. В большинстве случаев, если вы превысите лимит использования по тарифному плану Hobby, вам придётся подождать 30 дней , прежде чем снова воспользоваться функцией. Некоторые функции имеют более короткие или более длительные периоды действия: веб-аналитика.


1. Подготовить проект

Убедись, что проект работает локально.
Если это HTML + CSS + JS, то достаточно папки с файлами.
Если это React/Vue/Next.js и т.п., нужно настроить сборку (обычно npm run build создаёт папку dist или build).

2. Создать репозиторий

Залей проект на GitHub (или GitLab/Bitbucket).

3. Деплой на Vercel

Перейди на https://vercel.com
Войти через GitHub.
Нажми New Project → выбери свой репозиторий.
Vercel автоматически определит фреймворк (React, Next.js, Vue и т.д.).
Нажми Deploy → через пару секунд будет выдана ссылка вида:
https://your-project.vercel.app

4. Деплой на Netlify

Перейди на https://netlify.com
Зарегистрируйся через GitHub.
Нажми Add new site → Import from Git.
Выбери свой репозиторий.
Укажи:
Build command (например: npm run build, если React/Vue; если чистый HTML — оставь пустым).
Publish directory (обычно build/ или dist/).
Нажми Deploy site → получишь ссылку вида:
https://your-project.netlify.app


5. (Опционально) Подключить домен

И Netlify, и Vercel позволяют:
использовать свой домен (например, mywebsite.com);
настроить SSL автоматически.

Разница:
Vercel чаще используют для проектов на Next.js (он их автор).
Netlify популярен для статических сайтов и проектов на React/Vue.
